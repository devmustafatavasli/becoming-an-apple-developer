# Becoming an Apple Developer

A structured 18-week journey to deepen my iOS development skills,
focusing on Swift 6, SwiftUI, modern concurrency, SwiftData, and the
latest Apple platform technologies introduced with iOS 26.

This repository documents the path — every week, every concept, every
project I build along the way.

## Structure

Each week lives in its own directory and contains Playgrounds, mini
projects, and notes covering that week's focus area.

## Tech Focus

- Swift 6.x (strict concurrency, typed throws)
- SwiftUI with the Observation framework
- SwiftData and CloudKit
- Foundation Models framework (on-device AI)
- Liquid Glass design system
- App Intents, WidgetKit, Live Activities
